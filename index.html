Hearing Test (Experimental)

This is a simple, experimental hearing test that plays pure tones across a wide frequency range. For each tone, the volume slowly increases from silence. Click "I hear it" as soon as you can perceive the sound, and the test will move to the next frequency.

How to use

  • • Use headphones (strongly recommended)
  • • Sit in a quiet environment
  • • Start at a comfortable system volume (do not use maximum volume)
  • • Respond as soon as you hear the tone, not when it becomes loud

Important disclaimer

  • • This is not a medical or diagnostic tool
  • • Results can be misleading due to many variables, including:
    • ◦ Headphones or speakers
    • ◦ Sound card / DAC
    • ◦ Browser and operating system
    • ◦ Ambient noise
  • • Results cannot be compared between different people if different hardware is used
  • • The test is intended only for basic, personal experimentation, such as comparing left vs. right ear on the same setup

If you have concerns about your hearing, consult a qualified hearing professional.
